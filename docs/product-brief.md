# Dystn — Product Brief

## What is it?

Dystn is a Danish party game platform for friends and family playing together in the same room. One person hosts the game on a big screen (TV/laptop), and everyone else joins on their phones using a simple room code. No app install, no login — just open a link and play.

## Who is it for?

Groups of 1–8 people at parties, family gatherings, or casual hangouts who want a fun shared experience without needing to buy or install anything.

## How does it work?

1. The host creates a room and gets a 4-letter code
2. Players go to the website on their phones and enter the code + a name
3. The host picks a game from the lobby
4. Everyone plays together — phones are used for input, the big screen shows the action
5. After a game ends, the host can play again or switch to a different game — no re-joining needed

## The Games

### Duel
Write funny answers to the same question. Everyone votes on the best one. Think quick, be funny — the crowd decides the winner.

### Bluff
A trivia question with a missing word. Write a fake answer that sounds real. Then guess which answer is actually true among everyone's lies. Score points by fooling others.

### Tegn & Gæt (Draw & Guess)
Everyone draws a secret word on their phone. Others write fake guesses to trick people, then everyone votes on which guess is the real word.

### Telefon (Telephone)
Write a sentence. The next person draws it. The next person guesses the drawing. The next person draws that guess. At the end, the whole chain is revealed — see how hilariously wrong it went.

## Design Principles

- **Heads up, not heads down** — phones are for quick input only. The shared screen is where the fun happens.
- **Breathing room** — reveals are staggered one at a time with dramatic pauses, giving people time to laugh and react.
- **The room is the party** — create a room once, play all night. Players stay connected across multiple games.
- **No barriers for guests** — no accounts, no app download, no payment. Open the link and play. (Hosts unlock premium packs with a one-time purchase.)
- **Danish first** — all text is in Danish. This is built for Danish-speaking groups.
